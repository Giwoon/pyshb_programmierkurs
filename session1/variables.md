# Variablen
Variablen sind in Python eine dynamisch Typisiert. Das heißt der Inhalt und der Typ kann sich während der Laufzeit des Programms jederzeit ändern.

```python
>>> x = "Hallo Welt!"
>>> x
'Hallo Welt!'
>>> x = 42
>>> x
42
```

Wir weisen der Variable x erst ein Text zu und danach eine Zahl.


## Typen

### Integer

```python
>>> x = 40
>>> x + 2
42
type(x)
<class 'int'>
```

### Boolean

```python
>>> x = True
>>> x and False
False
type(x)
<class 'bool'>
```

### Float

```python
>>> x = 23.0
>>> x + .2
23.2
type(x)
<class 'float'>
```

### String

```python
>>> x = 'Hallo '
>>> x + 'Welt!'
'Hallo Welt'
type(x)
<class 'str'>
```
